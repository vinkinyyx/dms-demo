## 🎬 典型业务流程

1. 登录 → 自动进入 workspace.html 首页（M3 主题仪表盘）
2. 侧栏 → **销售订单** → 新建 → 选经销商 → 选产品（**仅显示授权产品**）→ 提交 → 审批 → **系统自动生成 CK 销售出库草稿**
3. 侧栏 → **销售出库** → 找到 DRAFT 单 → 点"✅ 执行发货" → 弹窗选批次/序列号 → 提交 → 库存扣减
4. 类似流程处理**采购订单 → 收货入库(RK)**（收货时手工录入批次）
5. 侧栏 → **销售岗位** → 左树选岗位 → 右侧看详情 → "绑定销售账号" / "挂载经销商"
