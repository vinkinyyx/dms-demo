## 🛠️ 部署过程摘要

1. ✅ 从 Windows 通过 plink/pscp 连接 SSH（无需 sshpass）
2. ✅ 修改 Docker daemon.json，配置 4 个 Docker Hub 镜像源（`hub.rat.dev` 命中）
3. ✅ 上传项目 zip 包（238KB 源码 + Dockerfile + Nginx conf + 首页）
4. ✅ 拉取镜像：postgres:14 / redis:7 / maven:3.9-17 / temurin:17-jre / nginx:1.25
5. ✅ 多阶段 Docker build（Maven 阿里云镜像加速依赖下载，约 5 分钟）
6. ✅ 三容器编排启动 → Flyway 自动执行 V1~V7 → Seed 全量数据
7. ✅ 生成正确的 bcrypt hash 修复 admin 密码（原 seed hash 是示例 hash）
8. ✅ Nginx 反代 /api/* → dms-backend，首页占位页展示

## ⚠️ 已知限制与提醒

1. **磁盘空间**：仅剩 2.8GB（93% 使用），需及时清理 Docker 构建缓存：
   ```bash
   docker system prune -a --volumes -f
   ```
2. **首次密码**：admin/Sh123456 是演示密码，生产环境**必须立即改**
3. **默认租户**：`tenantCode = "default"`（V1 单租户演示环境）
4. **前端占位**：目前是静态 HTML + 反代 Swagger，真正 Vue 前端待后续开发
5. **无 HTTPS**：仅 HTTP 明文；生产上线需配 Let's Encrypt 或阿里云证书
6. **无备份**：pgdata 存在 Docker volume；建议每天 cron pg_dump

---

## 🚀 下一步建议

1. **HTTPS + 域名**：申请域名 → 阿里云 SSL 证书 → Nginx 加 443
2. **前端开发**：Vue 3 + Vite + Element Plus / Vant（Monorepo 结构）
3. **CI/CD**：GitHub Actions 或 GitLab CI，自动 `docker build` + `docker push` + SSH 拉取部署
4. **监控**：Prometheus + Grafana + node_exporter（当前 2G 内存较紧张）
5. **日志聚合**：Loki + Promtail 或 ELK
6. **升级云服务器规格**：2G 内存 54% 已占用，跑真实业务需升到 4G+
